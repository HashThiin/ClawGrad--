# 02_超时响应与交互式退出机制

## 日期
2026-05-20

## 问题描述
AI 模型（尤其是长文本多题批改场景）响应时间不稳定，单次请求可能长达 2~3 分钟，多阶段流水线总耗时经常超过 5 分钟。原系统在 HTTP 超时时直接报错崩溃，用户没有任何选择余地。

## 修复方案

### 白话方案
当 AI 模型超时时，不要直接报错。做三件事：
1. **设一个 5 分钟的超时上限**
2. **超时后弹出一个友好界面**，告诉用户"思考时间过长"
3. **给用户三个选项**：换一个快速模型重试 / 返回重新配置 / 退出

后端检测到是超时异常就标记为 TIMEOUT 状态（不是 FAILED），并判断当前使用的模型是否已经是快速版（flash）。如果不是，建议切换。

### 代码解释

**递归检测异常链中是否含超时异常**

```java
// GradingPipeline.java
// 递归检查异常 cause 链中是否含 ReadTimeoutException 或 TimeoutException
private boolean isTimeoutException(Throwable t) {
    if (t == null) return false;
    if (t instanceof ReadTimeoutException || t instanceof TimeoutException) return true;
    return isTimeoutException(t.getCause());  // 递归深入 cause 链
}

// 流水线 catch 块
try {
    // ... 执行各阶段
} catch (Exception e) {
    if (isTimeoutException(e)) {
        // 超时 → 标记 TIMEOUT，判断是否需要建议快速模型
        boolean suggestFast = !model.contains("flash");
        taskStore.timeoutTask(taskId, e.getMessage(), suggestFast);
    } else {
        taskStore.failTask(taskId, e.getMessage());
    }
}
```

**TIMEOUT 状态 + suggestFastModel 字段**

```java
// GradingTaskStore.java
// TaskStatus 新增 suggestFastModel 字段
private boolean suggestFastModel;

public void timeoutTask(String taskId, String error, boolean suggestFastModel) {
    TaskStatus status = tasks.get(taskId);
    if (status != null) {
        status.setStatus("TIMEOUT");
        status.setError(error);
        status.setSuggestFastModel(suggestFastModel);  // true=建议换快速模型
        // ... 持久化任务
    }
}
```

**前端 TIMEOUT UI**

```tsx
// ResultPage.tsx
// 检测到 TIMEOUT 状态时显示交互界面
const checkResult = async () => {
    const res = await getTaskResult(taskId);
    if (res.status === 'TIMEOUT') {
        // 显示 TIMEOUT 界面，含已识别作业内容和操作按钮
        setTimedoutData({
            question: res.question,
            answer: res.answer,
            organizedHomework: res.organizedHomework,
            suggestFastModel: res.suggestFastModel
        });
    }
};

// 更换快速模型重试
const handleRetryWithFastModel = () => {
    // 用 qwen3.6-flash 重新提交任务
    submitGrading({ model: 'bailian-token-plan/qwen3.6-flash', ... });
};
```
